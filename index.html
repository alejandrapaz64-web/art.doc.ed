Conexión Google Sheets
📊
Conecte su hoja de Google Sheets
El sistema leerá los datos de estudiantes en tiempo real desde su Google Sheets. Siga estos pasos:
1
Abra su Google Sheets → Extensiones → Apps Script
Se abre el editor de scripts. Borre todo el contenido y pegue el código del archivo apps_script.gs que le entregué.
2
Implementar → Nueva implementación → Aplicación web
Ejecutar como: Yo · Acceso: Cualquier persona · Haga clic en Implementar y autorice.
3
Copie la URL del Web App y péguela arriba
La URL tiene el formato: https://script.google.com/macros/s/.../exec

Los datos se sincronizan automáticamente cuando agrega, edita o elimina estudiantes.
⚠️ Importante · Sincronización desde archivo local
Desde un archivo local, Chrome bloquea la sincronización automática por seguridad.
Solución 1: Use el botón de abajo para exportar manualmente.
Solución 2: Suba el sistema a GitHub Pages — desde ahí funciona automáticamente.
💡 El botón "Exportar al Sheets" envía todos los estudiantes registrados a su Google Sheets de una vez.
